# RAG System for TP-Link Archer C80 User Manual

## Deskripsi
Sistem RAG untuk jawab pertanyaan manual router tanpa halusinasi.

## Pembagian Tugas
- Raihan: Data ingestion dan chunking.
- Raihan: Embedding dan vector DB.
- Raihan: Retrieval dan generation.
- Raihan: Testing dan dokumentasi.

## Cara Run
1. Install dependencies: pip install transformers scipy numpy torch sentence-transformers
2. Run: python rag_router.py
3. Contoh: rag_query("Pertanyaanmu")

## Dependensi
- Transformers (HuggingFace)
- Numpy, Scipy, Torch
